# ProxyKiller-Admin-Notify
Simple module for [ProxyKiller](https://forums.alliedmods.net/showthread.php?t=313591) to notify in-game admins if a connecting player is utilizing a VPN/Proxy.

# Requirements
- [ProxyKiller](https://forums.alliedmods.net/showthread.php?t=313591) 2.0.0+

# Configuration
By default, players with the generic admin flag are notified in chat. This can be overridden using admin_overrides.cfg with `sm_proxykiller_printtoadmins`

# Installation
- Drag .smx into your /addons/sourcemod/plugins folder
- Drag translation file into your /addons/sourcemod/translations folder
